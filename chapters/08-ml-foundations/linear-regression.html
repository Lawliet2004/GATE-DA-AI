Linear Regression

GATE Relevance: ★★★ HIGH GATE 2022, 2023, 2024

Prerequisites

  • Supervised Learning Overview — loss functions, ERM
  • Linear Algebra — matrix multiplication, inverse, transpose
  • Calculus — partial derivatives, gradient vector

🎯 Learning Objectives

  • Derive the Normal Equation for ordinary least squares
  • Implement gradient descent for linear regression
  • Compute and interpret R², adjusted R², MSE, MAE
  • Apply Ridge (L2), Lasso (L1), and Elastic Net regularisation
  • Explain the bias-variance tradeoff quantitatively

1. Simple Linear Regression

Model

y = θ0 + θ1x + ε

where θ0 is the intercept (bias), θ1 is the slope, and ε ~ N(0, σ²) is noise.

Closed-Form Solution (Simple Case)

θ1 = ∑(xi − x̄)(yi − ȳ) / ∑(xi − x̄)²
θ0 = ȳ − θ1·x̄

2. Multiple Linear Regression

Model

y = Xθ + ε, where X is the n×(d+1) design matrix (with a column of 1s for the intercept), θ is the (d+1)×1 parameter vector.

2.1 The Normal Equation

Normal Equation Derivation

Objective: minimise J(θ) = ||y − Xθ||² = (y − Xθ)T(y − Xθ)

Expanding: J(θ) = yTy − 2θTXTy + θTXTXθ

Taking the gradient and setting to zero:

∇θJ = −2XTy + 2XTXθ = 0
θ* = (XTX)−1XTy

This requires XTX to be invertible (full column rank). Complexity: O(nd² + d³).

⚠ When the Normal Equation Fails

  • XTX is singular (multicollinearity) — use regularisation or pseudo-inverse
  • d (features) is very large — matrix inversion O(d³) becomes prohibitive; use gradient descent instead

💡 Worked Example 2 — Normal Equation (2 features)

Data (with bias column):

X = [[1,1,1],[1,2,3],[1,3,2],[1,4,5]], y = [2,5,4,8]

Step 1: XTX = [[4,10,11],[10,30,30],[11,30,39]]

Step 2: XTy = [19,55,57]

Step 3: Solve (XTX)θ = XTy

Using elimination or inverse: θ ≈ [−0.5, 1.0, 0.75]

Model: ŷ = −0.5 + 1.0·x1 + 0.75·x2

3. Gradient Descent

Batch Gradient Descent Update Rule

θj := θj − α · (1/n) ∑i=1n (hθ(xi) − yi) · xi,j

where α is the learning rate.

Algorithm: Batch Gradient Descent for Linear Regression

Initialize θ = [0, 0, ..., 0]
Repeat until convergence:
    For each parameter j = 0, 1, ..., d:
        gradient_j = (1/n) * ∑_{i=1}^{n} (h(x_i) - y_i) * x_{i,j}
        θ_j = θ_j - α * gradient_j
    End For
End Repeat

💡 Worked Example 3 — One Step of Gradient Descent

Data: (1,1), (2,2), (3,3). Model: ŷ = θ0 + θ1x. Initial: θ0=0, θ1=0. Learning rate α=0.1.

Predictions: h(1)=0, h(2)=0, h(3)=0

Errors: (0−1)=−1, (0−2)=−2, (0−3)=−3

Gradient for θ0: (1/3)(−1+−2+−3)·1 = −2

Gradient for θ1: (1/3)(−1·1+−2·2+−3·3) = (1/3)(−14) = −4.667

Update: θ0 = 0 − 0.1(−2) = 0.2, θ1 = 0 − 0.1(−4.667) = 0.467

After 1 step: ŷ = 0.2 + 0.467x

📝 Learning Rate Selection

  • Too large α → divergence (cost oscillates or increases)
  • Too small α → very slow convergence
  • Practical: start with α = 0.01, try powers of 10
  • Feature scaling (standardization) helps convergence dramatically

4. Evaluation Metrics

4.1 Mean Squared Error (MSE)

MSE = (1/n) ∑i=1n (yi − ŷi)²

4.2 Mean Absolute Error (MAE)

MAE = (1/n) ∑i=1n |yi − ŷi|

4.3 R-squared (Coefficient of Determination)

R² Formula

R² = 1 − SSres/SStot = 1 − ∑(yi−ŷi)² / ∑(yi−ȳ)²

Interpretation: the proportion of variance in y explained by the model. R² ∈ [0,1] for training data (can be negative on test data if the model is worse than predicting the mean).

4.4 Adjusted R²

R²adj = 1 − [(1−R²)(n−1)/(n−d−1)]

Penalises adding features that don’t improve the model. Preferred for comparing models with different numbers of features.

💡 Worked Example 4 — R² Calculation

Actual: [3, 5, 7, 9], Predicted: [2.8, 5.2, 6.8, 9.2], Mean ȳ = 6

SSres = (0.2)²+(−0.2)²+(0.2)²+(−0.2)² = 0.04+0.04+0.04+0.04 = 0.16

SStot = (−3)²+(−1)²+(1)²+(3)² = 9+1+1+9 = 20

R² = 1 − 0.16/20 = 1 − 0.008 = 0.992 (excellent fit!)

MSE = 0.16/4 = 0.04

5. Regularisation

Regularisation adds a penalty to prevent overfitting by discouraging large weights.

5.1 Ridge Regression (L2)

Ridge Objective

Jridge(θ) = ∑(yi−Xiθ)² + λ·||θ||2² = ||y−Xθ||² + λθTθ
θridge = (XTX + λI)−1XTy

The term λI ensures invertibility even when XTX is singular. λ controls regularisation strength.

5.2 Lasso Regression (L1)

Lasso Objective

Jlasso(θ) = ∑(yi−Xiθ)² + λ·||θ||1 = ||y−Xθ||² + λ∑|θj|

Key property: Lasso produces sparse solutions (many weights become exactly zero) — acts as automatic feature selection. No closed-form solution; solved via coordinate descent or LARS.

5.3 Elastic Net

Jelastic(θ) = ||y−Xθ||² + λ1||θ||1 + λ2||θ||2²

Combines L1 and L2 penalties. Useful when features are highly correlated (Lasso may arbitrarily pick one, Elastic Net groups them).

💡 Worked Example 5 — Ridge with λ

XTX = [[2,1],[1,2]], XTy = [5,4], λ = 1

XTX + λI = [[3,1],[1,3]]

Inverse of [[3,1],[1,3]] = (1/8)[[3,−1],[−1,3]]

θridge = (1/8)[[3,−1],[−1,3]] · [5,4] = (1/8)[11,7] = [1.375, 0.875]

Compare without regularisation: (XTX)−1XTy = (1/3)[[2,−1],[−1,2]]·[5,4] = [2.0, 1.0]

Ridge shrinks both coefficients toward zero.

6. Bias-Variance Tradeoff

Bias-Variance Decomposition

E[(y − h(x))²] = Bias(h)² + Variance(h) + σ²noise
  • Bias: Error from wrong assumptions. High bias → underfitting.
  • Variance: Error from sensitivity to training data fluctuations. High variance → overfitting.
  • Irreducible error (σ²): Noise in data; cannot be reduced.

Low (e.g., linear)HighLowHighHigh

High (e.g., degree-20 poly)LowHighVery lowVery high

📝 GATE Tip

Regularisation parameter λ: as λ increases, bias increases and variance decreases. The optimal λ minimises the total test error. Cross-validation is used to find this λ.

7. Assumptions of Linear Regression

📝 Classical Assumptions (Gauss-Markov)

  1. Linearity: y = Xθ + ε (linear in parameters)
  2. Independence: Observations are independent
  3. Homoscedasticity: Var(εi) = σ² (constant variance)
  4. No multicollinearity: Features are not perfectly correlated
  5. Normality of errors: ε ~ N(0, σ²) (for hypothesis testing and confidence intervals)

⚠ GATE Trap

Linear regression assumes linearity in parameters, not in features. You can have y = θ0 + θ1x + θ2x² — this is still “linear regression” because it is linear in (θ0, θ1, θ2). The features can be any nonlinear transformation of x.

Summary

📚 Key Formulas to Remember

  • Normal Equation: θ* = (XTX)−1XTy
  • Ridge: θ = (XTX + λI)−1XTy
  • GD update: θ := θ − α∇J(θ)
  • R² = 1 − SSres/SStot
  • Bias-Variance: E[Error] = Bias² + Variance + Noise
