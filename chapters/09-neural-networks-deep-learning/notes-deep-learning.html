Chapter 9 — Quick Reference Notes

📑 Key Formulas

Perceptron Update

y = sign(w · x + b)
w ← w + η(ytrue − ypred)x

MLP Forward Pass

z[l] = W[l] a[l−1] + b[l]
a[l] = f(z[l])

Backpropagation (Output Layer)

δ[L] = ∇aJ ⊙ f'(z[L])
∂J/∂W[l] = δ[l] (a[l−1])T
δ[l−1] = (W[l])T δ[l] ⊙ f'(z[l−1])

CNN Output Size

O = ⌊(W − F + 2P) / S⌋ + 1

Attention

Attention(Q,K,V) = softmax(QKT / √dk) V

Adam Update

mt = β1 mt−1 + (1−β1) gt
vt = β2 vt−1 + (1−β2) gt²
θ ← θ − η m̂t / (√v̂t + ε)

⚠ Common GATE Mistakes

  • Forgetting the +1 (bias) in parameter counting.
  • Confusing floor vs ceiling in CNN output formula — always use floor.
  • Thinking pooling has learnable parameters — it has zero.
  • Applying dropout at test time — only used during training.
  • Mixing up LSTM gates: forget gate erases; input gate writes.
  • Forgetting bias correction in Adam (÷ by 1−βt).
  • Ignoring the scaling factor √dk in attention.
