GATE Relevance: Critical

📅 One Week Before the Exam

1. Search Algorithms

  • Review the BFS / DFS / IDDFS complexity table — know Time, Space, Completeness, and Optimality for each strategy with branching factor b and depth d.
  • Practice A* search with two different heuristics on the same graph. Verify f(n) = g(n) + h(n) ordering and confirm the optimal path in both cases.
  • Trace alpha-beta pruning on a game tree with at least 8 leaf nodes. Identify which branches get pruned and confirm the minimax value at the root.

2. Logic

  • Revise every step of CNF conversion: eliminate biconditionals, eliminate implications, push negation inward (De Morgan’s), distribute OR over AND.
  • Practice 3 full resolution proofs from scratch — convert KB to CNF, negate the query, apply resolution until you derive the empty clause or exhaust clauses.
  • Review Skolemization rules: remove existential quantifiers by introducing Skolem constants (no preceding universal) or Skolem functions (with preceding universals).
  • Practice unification: solve at least 3 unification problems. Remember the occurs check — you cannot unify X with f(X).

3. Probability & Bayesian Networks

  • Solve 5 Bayes’ theorem problems covering medical diagnosis, spam filtering, and conditional independence scenarios.
  • Practice variable elimination on a 4-node Bayesian Network. Write out every factor, perform the summation steps, and normalize the result.
  • Review d-separation rules: a path is blocked if (a) a chain/fork node is observed, or (b) a collider node (and all descendants) are NOT observed. Practice on 3 different network topologies.

4. Machine Learning

  • Know the bias-variance tradeoff diagram cold: high bias = underfitting, high variance = overfitting, sweet spot is in between. Be able to sketch it.
  • Memorize evaluation metric formulas: Accuracy, Precision, Recall, F1-score, Specificity, AUC interpretation, MSE, MAE, R².
  • Review regularization: L1 (Lasso) promotes sparsity, L2 (Ridge) shrinks weights uniformly. Know the cost function form: J(θ) + λ∑|θ| vs J(θ) + λ∑θ².

5. Neural Networks

  • Practice backpropagation by hand on a 2-layer network (input → hidden → output). Compute forward pass, loss, then ∂L/∂w for each weight using the chain rule.
  • Know activation functions and their derivatives: Sigmoid σ(x) = 1/(1+e−x), σ′ = σ(1−σ); Tanh; ReLU max(0,x), ReLU′ = 0 or 1; Softmax for multi-class.
  • CNN output size formula: O = (W − F + 2P) / S + 1. Practice with at least 2 different configurations of W (input), F (filter), P (padding), S (stride).

6. Reinforcement Learning

  • Solve 3 Q-learning update problems: Q(s,a) ← Q(s,a) + α[R + γ maxa′ Q(s′,a′) − Q(s,a)]. Do the arithmetic step by step.
  • Know Bellman equations cold: V(s) = maxa ∑s′ P(s′|s,a)[R(s,a,s′) + γV(s′)] and Q(s,a) = ∑s′ P(s′|s,a)[R + γ maxa′ Q(s′,a′)].
  • Understand policy iteration vs value iteration: Policy iteration alternates evaluation and improvement until stable. Value iteration applies Bellman optimality update until convergence, then extracts policy once.

7. Constraint Satisfaction Problems

  • Practice arc consistency (AC-3) on a small CSP (≤4 variables). For each arc (Xi, Xj), remove values from Di that have no support in Dj. Re-add affected arcs to the queue.
  • Know backtracking search heuristics: MRV (Minimum Remaining Values) for variable selection, LCV (Least Constraining Value) for value ordering, and forward checking for early pruning.

8. Planning

  • Review STRIPS representation: each action has Preconditions, Add List (positive effects), and Delete List (negative effects). Practice writing STRIPS operators for a blocks-world problem.
  • Practice mutex identification in planning graphs: two actions are mutex if they have inconsistent effects, interference, or competing needs. Two literals are mutex if every way to achieve them is pairwise mutex.

🌙 Day Before the Exam

  • Review the Formula Sheet end to end. Read every formula aloud once and write down the 5 you find hardest to remember.
  • Go through Common Mistakes — focus on the mistakes you made during practice. Mark any that still feel tricky.
  • Take Mock Test 3 under strict timed conditions (25 minutes). Review every wrong answer and understand why you got it wrong.
  • Prepare your admit card, a valid photo ID (matching the admit card name exactly), and print/save directions to the exam center.
  • Get a minimum of 7 hours of sleep. Set two alarms. No late-night cramming — sleep consolidates memory far better than last-minute reading.
  • Prepare a water bottle and light snacks (energy bar, fruit) for the break. Lay out comfortable clothes and a watch (if allowed).

🏋 Exam Day Strategy

  • Arrive at the exam center at least 30 minutes early. Use the extra time to settle in, use the restroom, and calm your nerves.
  • First pass (15–20 min): Quickly scan all questions. Attempt every question you are confident about. Mark uncertain ones for later and move on — do not get stuck.
  • Second pass (40–50 min): Tackle medium-difficulty questions. These are the ones where you know the approach but need to work through the computation or reasoning carefully.
  • Third pass (remaining time): Attempt hard questions and review all marked/flagged answers. Double-check any answer you were unsure about.
  • NAT questions: Double-check your arithmetic. Re-read the question to confirm units and rounding requirements. A silly calculation error can cost you 2 marks.
  • MSQ questions: Remember there is partial marking — only select options you are genuinely sure about. Selecting a wrong option costs you marks, so be conservative.
  • MCQ questions: Do NOT leave MCQs unanswered. With 1/3 negative marking, if you can eliminate 2 out of 4 options, the expected value of guessing is positive. Always make an educated guess.

⚡ Key Formulas — Quick Recall (5-Minute Revision)

Scan through these 20 essential formulas one last time. No explanations — you should already know what each one means.

  1. BFS complexity: O(bd) time, O(bd) space
  2. DFS complexity: O(bm) time, O(bm) space
  3. IDDFS complexity: O(bd) time, O(bd) space
  4. A* evaluation: f(n) = g(n) + h(n)
  5. Minimax: V(s) = maxa mins′ V(s′)
  6. Bayes’ theorem: P(A|B) = P(B|A)·P(A) / P(B)
  7. Chain rule (probability): P(A,B,C) = P(A)·P(B|A)·P(C|A,B)
  8. Entropy: H(X) = −∑ p(x) log⊂2 p(x)
  9. Information gain: IG(S,A) = H(S) − ∑v (|Sv|/|S|) · H(Sv)
  10. Linear regression (normal eq.): θ = (XTX)−1XTy
  11. Gradient descent: θ ← θ − α∇J(θ)
  12. Sigmoid: σ(z) = 1 / (1 + e−z)
  13. Softmax: P(y=k) = ezk / ∑j ezj
  14. Cross-entropy loss: L = −∑ yi log(ŷi)
  15. CNN output size: O = (W − F + 2P) / S + 1
  16. Q-learning update: Q(s,a) ← Q(s,a) + α[R + γ maxa′Q(s′,a′) − Q(s,a)]
  17. Bellman equation: V(s) = maxa ∑s′ P(s′|s,a)[R + γV(s′)]
  18. Precision: TP / (TP + FP)
  19. Recall: TP / (TP + FN)
  20. F1 Score: 2 · (Precision · Recall) / (Precision + Recall)
