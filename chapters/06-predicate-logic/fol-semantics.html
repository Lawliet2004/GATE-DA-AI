FOL Semantics — Meaning and Interpretation

GATE Relevance: High

Prerequisites

  • FOL Syntax — terms, atoms, WFFs, quantifiers
  • Propositional Logic Semantics — truth tables, models

🎯 Learning Objectives

  • Define an interpretation (structure) for FOL: domain + assignments
  • Evaluate truth of FOL formulas under a given interpretation
  • Distinguish satisfiable, valid, and unsatisfiable formulas
  • Understand logical consequence (φ ⊨ ψ) in FOL

1. Interpretations (Structures)

Definition — Interpretation

An interpretation (or structure) I for a FOL language consists of:

  1. Domain D — a non-empty set of objects (the “universe of discourse”).
  2. Constant assignment — each constant symbol c is mapped to an element cI ∈ D.
  3. Predicate assignment — each n-ary predicate P is mapped to a relation PI ⊆ Dn.
  4. Function assignment — each n-ary function f is mapped to a total function fI : Dn → D.
Variable Assignment

A variable assignment σ is a function that maps each variable to an element of D: σ : Variables → D.

We write σ[x ↦ d] for the assignment identical to σ except that x maps to d.

Example Interpretation

Language: constants {a, b}, predicate P (arity 1), function f (arity 1).

  • D = {1, 2, 3}
  • aI = 1,   bI = 2
  • PI = {1, 3}   (P is true for 1 and 3)
  • fI(1) = 2,   fI(2) = 3,   fI(3) = 1

2. Truth Under an Interpretation

Given interpretation I with domain D and variable assignment σ, the truth value of a formula φ is defined recursively:

Truth of Terms and Atoms
  • Term evaluation: val(c) = cI; val(x) = σ(x); val(f(t1,…,tn)) = fI(val(t1),…,val(tn)).
  • Atom: I,σ ⊨ P(t1,…,tn) iff (val(t1),…,val(tn)) ∈ PI
Truth of Compound Formulas
  • I,σ ⊨ ¬φ   iff   I,σ ⊭ φ
  • I,σ ⊨ φ ∧ ψ   iff   I,σ ⊨ φ and I,σ ⊨ ψ
  • I,σ ⊨ φ ∨ ψ   iff   I,σ ⊨ φ or I,σ ⊨ ψ
  • I,σ ⊨ φ → ψ   iff   I,σ ⊭ φ or I,σ ⊨ ψ
  • I,σ ⊨ ∀x φ   iff   for every d ∈ D: I,σ[x↦d] ⊨ φ
  • I,σ ⊨ ∃x φ   iff   for some d ∈ D: I,σ[x↦d] ⊨ φ

3. Satisfiability, Validity, Logical Consequence

Key Semantic Concepts
ConceptDefinition
Satisfiableφ is satisfiable if there exists an interpretation I and assignment σ such that I,σ ⊨ φ
Valid (Tautology)φ is valid (⊨ φ) if for every interpretation I and every σ: I,σ ⊨ φ
Unsatisfiableφ is unsatisfiable if no interpretation makes it true
Logical ConsequenceΓ ⊨ φ if every interpretation satisfying all formulas in Γ also satisfies φ
Logical Equivalenceφ ≡ ψ if φ ⊨ ψ and ψ ⊨ φ
Theorem — Validity and Unsatisfiability

φ is valid   iff   ¬φ is unsatisfiable.

Proof sketch: φ is valid means I,σ ⊨ φ for all I,σ. So I,σ ⊭ ¬φ for all I,σ. Hence ¬φ has no satisfying interpretation, i.e., ¬φ is unsatisfiable. □

FOL vs. Propositional Logic

In propositional logic, validity is decidable (truth tables). In FOL, validity is only semi-decidable: if φ is valid, a proof can be found; if φ is not valid, the search may never terminate. This is a consequence of Church’s theorem.

4. Model Theory Basics

Model

An interpretation I is a model of a sentence φ (or a set of sentences Γ) if I ⊨ φ (or I ⊨ every γ ∈ Γ).

Key results from model theory relevant to AI:

  • Compactness Theorem: A set of sentences Γ has a model iff every finite subset of Γ has a model.
  • Löwenheim–Skolem Theorem: If a countable set of FOL sentences has an infinite model, it has a model of every infinite cardinality.
GATE Relevance

GATE typically tests your ability to evaluate formulas under small given interpretations and determine satisfiability/validity. Deep model theory is rarely tested, but understanding models is crucial.

5. Worked Examples

Example 1 — Evaluating ∀x P(x) → P(a)

Claim: This formula is valid (true in every interpretation).

Proof: Let I be any interpretation with domain D. Let aI = d0 ∈ D.

  • Case 1: I ⊨ ∀x P(x). Then P holds for every element of D, including d0. So I ⊨ P(a). The implication is T → T = T.
  • Case 2: I ⊭ ∀x P(x). Then the antecedent is false, so the implication is vacuously true (F → anything = T).

In both cases the formula is true. ∴ It is valid. □

Example 2 — Evaluating under a Specific Interpretation

Given: D = {1, 2},   aI = 1,   PI = {1},   QI = {2}

Formula: ∀x (P(x) ∨ Q(x))

Evaluation:

  • x = 1: P(1) ∨ Q(1) = T ∨ F = T ✓
  • x = 2: P(2) ∨ Q(2) = F ∨ T = T ✓

Both elements satisfy P(x) ∨ Q(x), so ∀x (P(x) ∨ Q(x)) is true under this interpretation.

Example 3 — Showing a Formula is NOT Valid

Formula: ∃x P(x) → ∀x P(x)

Counter-interpretation: D = {1, 2}, PI = {1}.

  • ∃x P(x): True (P(1) is true).
  • ∀x P(x): False (P(2) is false).
  • T → F = F.

The formula is false in this interpretation, so it is not valid. However, it is satisfiable (e.g., D = {1}, PI = {1} makes it true).

Summary

  • An interpretation specifies a domain D plus mappings for constants, predicates, and functions.
  • Truth of quantified formulas: ∀x checks all domain elements; ∃x checks at least one.
  • Valid ⇔ true in all interpretations ⇔ ¬φ is unsatisfiable.
  • Γ ⊨ φ means φ is true in every model of Γ.
  • FOL validity is semi-decidable (not decidable) — unlike propositional logic.
