Probability Fundamentals

GATE Relevance: High

Prerequisites

  • Basic set theory (unions, intersections, complements)
  • Arithmetic and algebraic manipulation

🎯 Learning Objectives

  • Define sample space, events, and the axioms of probability
  • Apply addition and complement rules to compute event probabilities
  • Use permutations and combinations for counting-based problems
  • Solve GATE-style numerical probability problems

1. Sample Space and Events

Sample Space (S or Ω)

The sample space is the set of all possible outcomes of a random experiment.

Example — Dice Roll

Rolling a single fair die: S = {1, 2, 3, 4, 5, 6}. Each outcome is equally likely with probability 1/6.

Event

An event is any subset of the sample space. For example, "rolling an even number" is the event E = {2, 4, 6}.

Events can be combined using set operations:

  • Union (A ∪ B): A or B (or both) occurs
  • Intersection (A ∩ B): Both A and B occur
  • Complement (Ac): A does not occur
  • Mutually exclusive: A ∩ B = ∅

2. Axioms of Probability (Kolmogorov)

Kolmogorov Axioms

  1. Non-negativity: P(A) ≥ 0 for every event A
  2. Normalization: P(S) = 1
  3. Countable additivity: For mutually exclusive events A1, A2, …
    P(A1 ∪ A2 ∪ …) = P(A1) + P(A2) + …

Key Consequences

  • 0 ≤ P(A) ≤ 1 for every event A
  • P(∅) = 0
  • P(Ac) = 1 − P(A)

3. Addition Rule and Complement Rule

General Addition Rule:
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

When A and B are mutually exclusive (A ∩ B = ∅):

P(A ∪ B) = P(A) + P(B)
Complement Rule:
P(Ac) = 1 − P(A)

Common GATE Trap

Students often forget to subtract P(A ∩ B) in the addition rule. Always check if events overlap before adding probabilities directly.

Example 1 — Cards

A card is drawn from a standard 52-card deck. Find P(Heart ∪ King).

Solution: P(Heart) = 13/52, P(King) = 4/52, P(Heart ∩ King) = 1/52 (King of Hearts).
P(Heart ∪ King) = 13/52 + 4/52 − 1/52 = 16/52 = 4/13.

4. Counting: Permutations and Combinations

Permutation

The number of ways to arrange r objects from n distinct objects (order matters):

P(n, r) = n! / (n − r)!

Combination

The number of ways to choose r objects from n distinct objects (order does not matter):

C(n, r) = n! / [r! × (n − r)!]

Example 2 — Committee Selection

A committee of 3 is chosen from 10 people. How many ways?

Solution: C(10, 3) = 10! / (3! × 7!) = (10 × 9 × 8) / (3 × 2 × 1) = 120.

Example 3 — Word Arrangements

How many distinct arrangements of the letters in "GATE"?

Solution: 4 distinct letters ⇒ P(4, 4) = 4! = 24.

5. Worked Examples

Example 4 — Two Dice GATE Style NAT

Two fair dice are rolled. What is the probability that the sum is 7?

Solution: Total outcomes = 6 × 6 = 36. Favourable outcomes: (1,6), (2,5), (3,4), (4,3), (5,2), (6,1) = 6.
P(sum = 7) = 6/36 = 1/6 ≈ 0.1667.

Example 5 — At Least One GATE Style MCQ

Three coins are tossed. What is P(at least one head)?

Solution: Use complement: P(at least one head) = 1 − P(no heads) = 1 − P(TTT) = 1 − (1/2)3 = 1 − 1/8 = 7/8.

Example 6 — Birthday Problem (Simplified)

In a group of 3 people, find P(at least two share a birthday) assuming 365 equally likely days.

Solution: P(no shared birthday) = (365/365) × (364/365) × (363/365) ≈ 0.9918.
P(at least one shared) = 1 − 0.9918 = ≈ 0.0082.

Example 7 — Defective Items GATE 2019

A lot contains 10 items, of which 3 are defective. If 4 items are drawn at random without replacement, what is P(exactly 2 defective)?

Solution: Using hypergeometric distribution:
P = C(3,2) × C(7,2) / C(10,4) = (3 × 21) / 210 = 63/210 = 3/10 = 0.30.

Summary

  • A sample space S contains all possible experiment outcomes; an event is a subset of S.
  • Kolmogorov axioms: non-negativity, normalization, and countable additivity.
  • Addition rule: P(A ∪ B) = P(A) + P(B) − P(A ∩ B).
  • Complement rule: P(Ac) = 1 − P(A).
  • Use permutations when order matters, combinations when it does not.
  • "At least one" problems — always consider the complement approach.
