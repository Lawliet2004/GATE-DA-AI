D-Separation and Conditional Independence in BNs

GATE Relevance: Very High

Prerequisites

  • Bayesian Networks Introduction
  • Conditional Independence

🎯 Learning Objectives

  • Identify the three canonical structures: chain, fork, collider
  • Apply the d-separation criterion to determine conditional independence
  • Explain the "explaining away" phenomenon
  • Define and compute the Markov blanket of a node

1. Three Canonical Structures

Chain: A → B → C

Without evidence on B: A and C are dependent (information flows through B).

Given B: A and C are conditionally independent (A ⊥ C | B). The path is blocked.

Example — Chain

Low Pressure → Rain → Traffic. Knowing it rained blocks the influence of low pressure on traffic. If we already know it rained, learning about low pressure tells us nothing new about traffic.

Fork: A ← B → C

Without evidence on B: A and C are dependent (common cause B induces correlation).

Given B: A and C are conditionally independent (A ⊥ C | B). The path is blocked.

Example — Fork

Shoe Size ← Age → Reading Level. Children with big feet tend to read better (confounded by age). Given age, shoe size and reading level are independent.

Collider (V-structure): A → B ← C

Without evidence on B (or its descendants): A and C are independent.

Given B (or any descendant of B): A and C become dependent. The path is unblocked.

Example — Collider (Explaining Away) GATE Favorite

Burglary → Alarm ← Earthquake. Without knowing the alarm status, burglary and earthquake are independent. But if the alarm rings and we learn there was an earthquake, the probability of burglary decreases — the earthquake "explains away" the alarm.

2. D-Separation Criterion

D-Separation

Two sets of variables X and Y are d-separated by a set Z if every undirected path between any node in X and any node in Y is blocked given Z.

A path is blocked if there exists a node W on the path such that:

  • W is in Z and the connection through W is a chain or fork, OR
  • W is a collider on the path and neither W nor any descendant of W is in Z

D-Separation ⇒ Conditional Independence

If X and Y are d-separated by Z, then X ⊥ Y | Z in every distribution that the BN can represent.

Example 1 — Student Network GATE Style

D → G, I → G, I → S, G → L. Is D ⊥ S | I?

Paths from D to S: D → G ← I → S. Node G is a collider. G is not observed. So this path is blocked.

No other paths exist. D ⊥ S | I? Yes.

Example 2 — Is D ⊥ S | G?

Path: D → G ← I → S. G is a collider and is observed, so the path is unblocked.

D ⊥ S | G? No — observing the grade creates a dependency between difficulty and SAT.

Example 3 GATE 2018

Given BN: A → C, B → C, C → D, C → E. Is A ⊥ B? Is A ⊥ B | D?

A ⊥ B: Path A → C ← B. C is a collider, not observed ⇒ blocked ⇒ A ⊥ B. Yes.

A ⊥ B | D: Path A → C ← B. C is a collider. D is a descendant of C and is observed ⇒ unblocked ⇒ A ¬⊥ B | D. No.

Example 4 — Longer Path

BN: X → A → B ← C → Y. Given {A}, is X ⊥ Y?

Path: X → A → B ← C → Y. A is observed (chain), so X–A is blocked. Path is blocked ⇒ X ⊥ Y | A. Yes.

3. Markov Blanket

Markov Blanket

The Markov blanket of a node X consists of:

  • X's parents
  • X's children
  • The other parents of X's children (co-parents)

Given its Markov blanket, X is conditionally independent of all other nodes in the network.

Example — Alarm Network

B → A, E → A, A → J, A → M. Markov blanket of A = {B, E, J, M}.

Markov blanket of B = {A, E} (parent of A, plus co-parent E).

Summary

  • Chain/Fork: blocked when middle node is observed
  • Collider: blocked when collider (and its descendants) are NOT observed
  • D-separation determines conditional independence in BNs — a core GATE topic
  • The Markov blanket fully shields a node from the rest of the network
  • Explaining away: observing a collider's effect creates dependence between its causes
