Joint Distributions

GATE Relevance: Medium

Prerequisites

  • Random Variables & Distributions
  • Independence

🎯 Learning Objectives

  • Define joint PMF and joint PDF
  • Compute marginal distributions from joint distributions
  • Calculate covariance and correlation coefficient
  • Understand the multivariate Gaussian at a high level

1. Joint PMF and PDF

Joint PMF (Discrete)

p(x, y) = P(X = x, Y = y),   ∑x∑y p(x, y) = 1

Joint PDF (Continuous)

f(x, y) ≥ 0,   ∫∫ f(x, y) dx dy = 1

Example 1 — Joint PMF Table

Two dice are rolled. X = min(die1, die2), Y = max(die1, die2). The joint PMF table has 21 unique (x, y) pairs with x ≤ y.

For x = y = 3: only one outcome (3,3), so p(3,3) = 1/36.
For x = 2, y = 5: outcomes (2,5) and (5,2), so p(2,5) = 2/36 = 1/18.

2. Marginal Distributions

Marginalization

pX(x) = ∑y p(x, y)   |   fX(x) = ∫ f(x, y) dy

To get the marginal of one variable, sum (or integrate) out the other variable.

Example 2 — Marginal from Table GATE Style

Given joint PMF table for X ∈ {0,1} and Y ∈ {0,1}:

Y=0Y=1
X=00.20.1
X=10.30.4

P(X = 0) = 0.2 + 0.1 = 0.3, P(X = 1) = 0.3 + 0.4 = 0.7
P(Y = 0) = 0.2 + 0.3 = 0.5, P(Y = 1) = 0.1 + 0.4 = 0.5

3. Covariance and Correlation

Covariance

Cov(X, Y) = E[XY] − E[X]E[Y]

If Cov(X, Y) = 0, X and Y are uncorrelated (but not necessarily independent!).

Correlation Coefficient

ρ(X, Y) = Cov(X, Y) / (σX · σY),   −1 ≤ ρ ≤ 1

Example 3 — Computing Covariance

Using the joint table from Example 2:
E[X] = 0(0.3) + 1(0.7) = 0.7,   E[Y] = 0(0.5) + 1(0.5) = 0.5
E[XY] = 0·0(0.2) + 0·1(0.1) + 1·0(0.3) + 1·1(0.4) = 0.4
Cov(X,Y) = 0.4 − 0.7 × 0.5 = 0.4 − 0.35 = 0.05

Uncorrelated ≠ Independent

If X ~ Uniform(−1,1) and Y = X2, then Cov(X,Y) = 0 but X and Y are clearly dependent. Zero correlation only implies independence for jointly Gaussian variables.

4. Multivariate Gaussian (Overview)

Multivariate Normal Distribution

f(x) = (1 / ((2π)d/2 |Σ|1/2)) exp(−½(x − μ)T Σ−1(x − μ))

Parameterised by mean vector μ ∈ ℝd and covariance matrix Σ ∈ ℝd×d.

Key Properties

  • Marginals are also Gaussian
  • Conditionals are also Gaussian
  • Uncorrelated ⇔ Independent (only for jointly Gaussian)
  • Used extensively in Gaussian processes, Kalman filters, and many ML models

Summary

  • Joint distributions describe the probability over multiple variables simultaneously
  • Marginal distributions obtained by summing/integrating out variables
  • Cov(X,Y) measures linear relationship; ρ normalizes to [−1, 1]
  • Multivariate Gaussian: fully specified by mean vector and covariance matrix
