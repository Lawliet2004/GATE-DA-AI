Chapter 7 Quiz: Reasoning Under Uncertainty

20 Questions
Time: 00:00

Q1. MCQ

If P(A) = 0.6 and P(B) = 0.5 and A, B are independent, what is P(A ∪ B)?

Answer: B. P(A ∪ B) = 0.6 + 0.5 − 0.6(0.5) = 0.8.

Q2. MCQ

Bayes' theorem states P(A|B) =

Answer: C. Bayes' theorem: P(A|B) = P(B|A)P(A)/P(B).

Q3. NAT

Factories produce 30%, 45%, 25% of items with defect rates 3%, 4%, 5%. An item is defective. P(Factory 2) = ______ (round to 4 decimal places).

Answer: 0.4557. P(D) = 0.0395. P(F2|D) = 0.018/0.0395 = 0.4557.

Q4. MCQ

For a fair six-sided die, what is the entropy H(X)?

Answer: A. Uniform distribution has entropy log2(n) = log2(6) ≈ 2.585 bits.

Q5. True/False

KL divergence is symmetric: DKL(P||Q) = DKL(Q||P).

Answer: False. KL divergence is NOT symmetric.

Q6. MCQ

In a Bayesian network A → C ← B, with C not observed, which is true?

Answer: D. Collider C not observed ⇒ A ⊥ B.

Q7. MCQ

The Markov blanket of a node X in a BN includes:

Answer: B. Markov blanket = parents + children + co-parents of children.

Q8. NAT

MLE of P(Heads) after 20 flips with 14 heads = ______.

Answer: 0.70. MLE = k/n = 14/20 = 0.70.

Q9. MCQ

In Naive Bayes, the "naive" assumption is:

Answer: C. The key assumption is P(x1,...,xn|C) = ∏ P(xi|C).

Q10. True/False

In an HMM, the Forward algorithm has time complexity O(T · N²) where T is the sequence length and N is the number of states.

Answer: True. Both Forward and Viterbi run in O(T N²).

Q11. MCQ

Which inference method gives exact results in Bayesian networks?

Answer: B. Variable elimination is an exact inference method. The others are approximate sampling methods.

Q12. NAT

X takes values {1,2,3,4} with probabilities {1/2, 1/4, 1/8, 1/8}. H(X) = ______ bits.

Answer: 1.75. H = 0.5(1) + 0.25(2) + 0.125(3) + 0.125(3) = 1.75 bits.

Q13. MCQ

The Bellman optimality equation for value function is:

Answer: A. Bellman optimality: V*(s) = maxa ∑ T(s,a,s')[R(s,a,s') + γV*(s')].

Q14. MSQ

Select ALL correct statements about Laplace smoothing in Naive Bayes:

Answer: A, C. Laplace smoothing adds 1 to counts, preventing zeros. It doesn't guarantee better accuracy or eliminate priors.

Q15. True/False

If the prior is uniform, then MAP estimation reduces to MLE.

Answer: True. Uniform prior ⇒ log P(θ) is constant ⇒ MAP = MLE.

Q16. MCQ

The Viterbi algorithm solves which HMM problem?

Answer: D. Viterbi finds the most probable hidden state sequence (decoding).

Q17. NAT

BN: A → B, A → C. P(A=T) = 0.3, P(B=T|A=T) = 0.9, P(B=T|A=F) = 0.4, P(C=T|A=T) = 0.7, P(C=T|A=F) = 0.2. P(B=T, C=T) = ______.

Answer: 0.245. = 0.3(0.9)(0.7) + 0.7(0.4)(0.2) = 0.189 + 0.056 = 0.245.

Q18. MCQ

What is mutual information I(X;Y)?

Answer: C. I(X;Y) = H(X) − H(X|Y) = H(Y) − H(Y|X).

Q19. True/False

In a BN, observing a collider's descendant does NOT activate the path through the collider.

Answer: False. Observing a collider OR any of its descendants unblocks the path (explaining away effect).

Q20. MCQ

In value iteration for an MDP, the discount factor γ must satisfy:

Answer: B. Discount factor γ ∈ [0, 1). For γ = 1, convergence is only guaranteed with finite horizons.
