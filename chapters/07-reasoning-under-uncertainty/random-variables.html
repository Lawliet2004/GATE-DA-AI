Random Variables and Distributions

GATE Relevance: High

Prerequisites

  • Probability Basics
  • Basic calculus (integration for continuous distributions)

🎯 Learning Objectives

  • Differentiate discrete and continuous random variables
  • Understand PMF, PDF, and CDF
  • Compute expected value and variance
  • Know the key distributions: Bernoulli, Binomial, Poisson, Uniform, Normal, Exponential

1. Discrete vs Continuous Random Variables

Random Variable

A random variable X is a function from the sample space to real numbers: X: S → ℝ.

  • Discrete: Takes countable values (e.g., number of heads in n tosses)
  • Continuous: Takes values in an interval (e.g., height, temperature)

2. PMF, PDF, and CDF

PMF (Probability Mass Function) — Discrete

p(x) = P(X = x),   ∑x p(x) = 1

PDF (Probability Density Function) — Continuous

f(x) ≥ 0,   ∫−∞∞ f(x) dx = 1,   P(a ≤ X ≤ b) = ∫ab f(x) dx

CDF (Cumulative Distribution Function)

F(x) = P(X ≤ x)

For discrete: F(x) = ∑k ≤ x p(k). For continuous: F(x) = ∫−∞x f(t) dt.

GATE Note

For continuous random variables, P(X = a) = 0 for any specific value a. Only intervals have nonzero probability.

3. Expected Value and Variance

Expected Value:
E[X] = ∑x x · p(x)   (discrete)  |  E[X] = ∫ x · f(x) dx   (continuous)
Variance:
Var(X) = E[X2] − (E[X])2 = E[(X − μ)2]
Standard Deviation: σ = √Var(X)

Useful Properties

  • E[aX + b] = aE[X] + b
  • Var(aX + b) = a2Var(X)
  • If X, Y independent: E[XY] = E[X]E[Y], Var(X + Y) = Var(X) + Var(Y)

4. Common Distributions

Bernoulli(p)

X ∈ {0, 1}, P(X = 1) = p.

Mean: p   |   Variance: p(1 − p)

Binomial(n, p)

Number of successes in n independent Bernoulli trials.

P(X = k) = C(n, k) · pk · (1 − p)n−k

Mean: np   |   Variance: np(1 − p)

Example — Binomial GATE Style

A fair coin is tossed 10 times. P(exactly 6 heads)?

P = C(10,6)(0.5)6(0.5)4 = 210/1024 = 105/512 ≈ 0.2051

Poisson(λ)

Number of events in a fixed interval (rate = λ).

P(X = k) = e−λ λk / k!

Mean: λ   |   Variance: λ

Uniform(a, b)

Continuous, equally likely over [a, b].

f(x) = 1 / (b − a) for a ≤ x ≤ b

Mean: (a + b)/2   |   Variance: (b − a)2/12

Normal (Gaussian) N(μ, σ2)

f(x) = (1 / σ√(2π)) · exp(−(x − μ)2 / (2σ2))

Mean: μ   |   Variance: σ2

68-95-99.7 rule: ~68% within ±1σ, ~95% within ±2σ, ~99.7% within ±3σ.

Exponential(λ)

Time between Poisson events.

f(x) = λ e−λx for x ≥ 0

Mean: 1/λ   |   Variance: 1/λ2

Memoryless property: P(X > s + t | X > s) = P(X > t)

Summary

  • PMF for discrete, PDF for continuous; CDF applies to both
  • Var(X) = E[X2] − (E[X])2 — the shortcut formula is essential for GATE
  • Know the mean and variance formulas for all six distributions
  • Normal and Binomial are the most frequently tested in GATE
