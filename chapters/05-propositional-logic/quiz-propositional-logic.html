🧪 Chapter Quiz: Propositional Logic

📝 Quiz Instructions

This quiz contains 15 questions covering all topics in Propositional Logic. Question types include MCQ, MSQ (Multiple Select), NAT (Numerical Answer Type), and True/False — matching GATE exam patterns.

Select your answer and click Check Answer to see detailed explanations.

Question 1 MCQ

What is the truth value of the expression (P → Q) ∧ (Q → P) when P = T and Q = F?

Answer: (b) False

Explanation: With P = T, Q = F: P → Q = T → F = F. Since (P→Q) is False, the conjunction (P→Q) ∧ (Q→P) = F ∧ anything = F.

Note: (P→Q)∧(Q→P) ≡ P↔Q. When P = T, Q = F, P↔Q = F.

Question 2 MCQ

Which of the following is logically equivalent to ¬(P → Q)?

Answer: (b) P ∧ ¬Q

Explanation:

  • P → Q ≡ ¬P ∨ Q (material implication)
  • ¬(P → Q) ≡ ¬(¬P ∨ Q)
  • ≡ P ∧ ¬Q (De Morgan's law)

This is a very common GATE identity. The only way an implication is false is when the antecedent is true and the consequent is false.

Question 3 MCQ

Which of the following is a tautology?

Answer: (a) P → (Q → P) and (d) (P∧Q) → (P∨Q)

Both (a) and (d) are tautologies. However, since this is MCQ with one answer expected:

Primary answer: (a)

Explanation for (a): P → (Q → P). Using material implication: Q → P ≡ ¬Q ∨ P, so the whole formula becomes P → (¬Q ∨ P) ≡ ¬P ∨ ¬Q ∨ P ≡ ⊤ (since P ∨ ¬P is included).

Why (b) fails: Set P = F, Q = T: (F→T)→F = T→F = F.

Why (c) fails: Set P = T, Q = F: T→(T→F) = T→F = F.

(d) is also a tautology: If P∧Q is true, then both P and Q are true, so P∨Q is true. If P∧Q is false, the implication is vacuously true.

Question 4 True/False

The converse of P → Q is Q → P, and it is logically equivalent to the original implication.

Answer: False

Explanation: The converse of P → Q is indeed Q → P, but they are NOT logically equivalent. Counterexample: P = T, Q = F → P→Q = F but Q→P = T. The contrapositive ¬Q → ¬P is equivalent to P → Q, not the converse.

Question 5 MCQ

Applying De Morgan's law, ¬(P ∧ (Q ∨ R)) is equivalent to:

Answer: (b) ¬P ∨ (¬Q ∧ ¬R)

Explanation:

  • ¬(P ∧ (Q ∨ R))
  • = ¬P ∨ ¬(Q ∨ R)    [De Morgan's on ∧]
  • = ¬P ∨ (¬Q ∧ ¬R)    [De Morgan's on ∨]

Question 6 NAT

For a propositional formula with 4 variables, the total number of rows in its truth table is _____.

Answer: 16

Explanation: For n variables, there are 2n possible truth assignments. With 4 variables: 24 = 16 rows.

Question 7 MCQ

Which of the following is in Conjunctive Normal Form (CNF)?

Answer: (b) (P ∨ Q) ∧ (¬R ∨ S) ∧ (¬P)

Explanation: CNF is a conjunction (AND) of disjunctive clauses (ORs of literals).

  • (a) is DNF — OR of ANDs
  • (b) is CNF — AND of ORs: ✓ each clause contains only literals with ∨
  • (c) contains →, not in normal form at all
  • (d) has ¬ applied to a compound formula, not in normal form

Question 8 MCQ

Given: "If it rains, the ground is wet. The ground is not wet." Which inference rule allows you to conclude "It did not rain"?

Answer: (b) Modus Tollens

Explanation: Modus Tollens: From P → Q and ¬Q, conclude ¬P. Here, P = "it rains", Q = "ground is wet". Given P → Q and ¬Q, we conclude ¬P ("it did not rain").

Question 9 NAT

How many satisfying truth assignments does the formula (P ∨ Q) ∧ (¬P ∨ Q) have?

Answer: 1

Explanation:

  • P=T, Q=T: (T∨T)∧(F∨T) = T∧T = T ✓
  • P=T, Q=F: (T∨F)∧(F∨F) = T∧F = F
  • P=F, Q=T: (F∨T)∧(T∨T) = T∧T = T ✓
  • P=F, Q=F: (F∨F)∧(T∨F) = F∧T = F

Wait — there are actually 2 satisfying assignments. Let me recheck:

Actually the answer is 2: (T,T) and (F,T). The formula simplifies to Q by resolution.

Question 10 MCQ

Given the clauses {P ∨ Q} and {¬P ∨ R}, the resolvent is:

Answer: (a) Q ∨ R

Explanation: Resolution rule: From (α ∨ P) and (β ∨ ¬P), derive (α ∨ β). Here α = Q, β = R, complementary literal is P/¬P. The resolvent is Q ∨ R.

Question 11 True/False

The clause (¬P ∨ ¬Q ∨ R ∨ S) is a Horn clause.

Answer: False

Explanation: A Horn clause has at most one positive (non-negated) literal. The clause ¬P ∨ ¬Q ∨ R ∨ S has two positive literals (R and S). Therefore, it is NOT a Horn clause.

An equivalent implication form would be: P ∧ Q → R ∨ S, which has a disjunctive head — not allowed in Horn clauses.

Question 12 NAT

The number of distinct Boolean functions that can be defined on 2 propositional variables is _____.

Answer: 16

Explanation: With n variables, we have 2n rows in the truth table. Each output column can independently be T or F for each row. So total distinct functions = 22n. For n = 2: 222 = 24 = 16.

Question 13 MSQ

Which of the following are valid inference rules? (Select all that apply)

Answer: (a) and (c)

Explanation:

  • (a) Valid — Modus Ponens: P → Q, P ⊢ Q ✓
  • (b) Invalid — Affirming the Consequent: P → Q, Q ⊬ P ✗ (fallacy)
  • (c) Valid — Modus Tollens: P → Q, ¬Q ⊢ ¬P ✓
  • (d) Invalid — Denying the Antecedent: P → Q, ¬P ⊬ ¬Q ✗ (fallacy)

Question 14 MCQ

Which of the following statements about the SAT problem is correct?

Answer: (c) Horn-SAT can be solved in linear time

Explanation:

  • (a) Wrong: 2-SAT is in P (solvable in O(n+m) using SCC)
  • (b) Wrong: 3-SAT is NP-complete
  • (c) Correct: Horn-SAT is solvable in O(n) using unit propagation
  • (d) Wrong: 1-SAT is trivially in P (linear time)

Question 15 MCQ

In resolution-based theorem proving, to prove that a formula α follows from a knowledge base KB, we:

Answer: (b) Convert KB ∧ ¬α to CNF and derive the empty clause

Explanation: Resolution is a refutation-based method. To prove KB ⊨ α:

  1. Negate the goal: assume ¬α
  2. Add ¬α to the KB clauses
  3. Convert everything to CNF
  4. Repeatedly apply resolution
  5. If the empty clause □ is derived, KB ∧ ¬α is unsatisfiable, proving KB ⊨ α
