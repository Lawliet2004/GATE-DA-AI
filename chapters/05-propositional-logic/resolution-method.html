Resolution in Propositional Logic

GATE DA — Very High Importance

🎯 Learning Objectives

  • Convert formulas to clause form for resolution
  • Apply the resolution algorithm systematically
  • Perform refutation proofs (proof by contradiction)
  • Understand resolution completeness

1. Overview of Resolution

Resolution is a single, powerful inference rule that forms the basis of many automated theorem provers. It works on formulas in Conjunctive Normal Form (CNF) and uses proof by refutation: to prove that KB ⊨ α, we show that KB ∧ ¬α is unsatisfiable.

Resolution Proof Strategy (Refutation)

To prove that a knowledge base KB entails a conclusion α:

  1. Convert KB to CNF (set of clauses)
  2. Add ¬α (negation of goal) as additional clauses
  3. Repeatedly apply resolution to pairs of clauses
  4. If the empty clause (□) is derived → KB ⊨ α is proved
  5. If no new clauses can be derived → α cannot be proved from KB

2. Converting to Clause Form

A clause is a set of literals, representing their disjunction. A CNF formula is a set of clauses, representing their conjunction.

Example: Formula to Clause Set

Formula: (P ∨ Q) ∧ (¬P ∨ R) ∧ (¬Q ∨ ¬R ∨ S)

Clause set: { {P, Q}, {¬P, R}, {¬Q, ¬R, S} }

Each clause is a set of literals separated by commas (implicit ∨).

3. Resolution Algorithm

Algorithm: Propositional Resolution
function PL-RESOLUTION(KB, α):
    clauses ← set of clauses in CNF of (KB ∧ ¬α)
    new ← {}
    
    loop:
        for each pair (Cᵢ, Cⱼ) in clauses:
            resolvents ← PL-RESOLVE(Cᵢ, Cⱼ)
            if □ (empty clause) ∈ resolvents:
                return TRUE  // KB entails α
            new ← new ∪ resolvents
        
        if new ⊆ clauses:
            return FALSE  // KB does not entail α
        clauses ← clauses ∪ new

function PL-RESOLVE(Cᵢ, Cⱼ):
    // Find complementary literals
    for each literal l in Cᵢ:
        if ¬l ∈ Cⱼ:
            resolvent ← (Cᵢ - {l}) ∪ (Cⱼ - {¬l})
            return {resolvent}
    return {}

📝 Important Rules

  • Resolve one pair of complementary literals at a time.
  • If the resolvent contains a literal and its complement (e.g., P and ¬P), it is a tautology — discard it.
  • Remove duplicate literals within a clause (idempotent law).
  • The empty clause □ represents False (⊥).

4. Step-by-Step Resolution Examples

Example 1: Prove Q from {P, P → Q}

Step 1: Convert to clauses

  • P → Q becomes ¬P ∨ Q
  • KB clauses: {P}, {¬P, Q}
  • Negate goal: ¬Q → clause {¬Q}

Step 2: Resolution

StepClause 1Clause 2Resolved OnResolvent
1{¬P, Q}{¬Q}Q{¬P}
2{P}{¬P}P□ (empty)

Empty clause derived → KB ⊨ Q is proved. ✅

Example 2: Prove R from {P → Q, Q → R, P}

Clauses from KB + ¬R:

  1. {¬P, Q}   (from P → Q)
  2. {¬Q, R}   (from Q → R)
  3. {P}   (premise)
  4. {¬R}   (negation of goal)

Resolution steps:

StepResolveOnResult
5(2) + (4)R{¬Q}
6(1) + (5)Q{¬P}
7(3) + (6)P□

Empty clause derived → R is proved from KB. ✅

Example 3: Prove S from {P ∨ Q, ¬P ∨ R, ¬Q ∨ S, ¬R ∨ S}

Clauses:

  1. {P, Q}
  2. {¬P, R}
  3. {¬Q, S}
  4. {¬R, S}
  5. {¬S}   (negation of goal)

Resolution:

StepResolveOnResult
6(3) + (5)S{¬Q}
7(4) + (5)S{¬R}
8(2) + (7)R{¬P}
9(1) + (6)Q{P}
10(8) + (9)P□

Empty clause derived → S is proved. ✅

5. Resolution Completeness

Theorem: Resolution is Refutation-Complete

If a set of propositional clauses is unsatisfiable, then resolution will always eventually derive the empty clause □. However, resolution is not complete for generating all entailed clauses — it can only prove unsatisfiability via refutation.

📝 What This Means for GATE

Resolution can answer: "Does KB entail α?" by testing if KB ∧ ¬α is unsatisfiable. If resolution derives □, the answer is yes. If no more resolvents can be generated without □, the answer is no.

6. Common Pitfalls

⚠️ Watch Out

  • Don't forget to negate the goal before adding it to the clause set.
  • Resolve only one complementary pair per resolution step.
  • Discard tautological resolvents (those containing both P and ¬P).
  • Make sure the formula is in CNF before starting resolution.
  • Don't confuse KB ⊨ α (entailment) with KB ⊢ α (derivation). Resolution shows the former via refutation.
