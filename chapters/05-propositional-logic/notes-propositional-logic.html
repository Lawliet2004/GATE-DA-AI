📋 Quick Revision Notes: Propositional Logic

📝 How to Use These Notes

These are condensed revision notes covering all key formulas, laws, and facts for GATE preparation. Review these before the exam!

1. Connectives Truth Table Summary

PQ¬PP∧QP∨QP→QP↔Q

🧠 Mnemonics

  • AND (∧): True only when BOTH true — "strict gatekeeper"
  • OR (∨): False only when BOTH false — "generous gatekeeper"
  • IMPLIES (→): False ONLY when T→F — "a broken promise"
  • IFF (↔): True when SAME values — "agreement checker"

2. Operator Precedence

¬ > ∧ > ∨ > → > ↔

(Highest to Lowest)

🧠 Mnemonic: "Not And Or Implies Iff" — NAOII

→ is right-associative: P → Q → R means P → (Q → R)

⚠️ Common Fallacies (INVALID!)

  • Affirming the Consequent: P → Q, Q ⊬ P
  • Denying the Antecedent: P → Q, ¬P ⊬ ¬Q

5. Normal Forms

FormStructureConversion
CNFAND of ORs: (l₁∨l₂)∧(l₃∨l₄)∧...Distribute ∨ over ∧
DNFOR of ANDs: (l₁∧l₂)∨(l₃∧l₄)∨...Distribute ∧ over ∨

📝 CNF Conversion Steps

  1. Eliminate ↔: α ↔ β → (α→β) ∧ (β→α)
  2. Eliminate →: α → β → ¬α ∨ β
  3. Push ¬ inward (De Morgan's, Double Negation)
  4. Distribute ∨ over ∧

9. Common GATE Patterns

🎯 Most Frequently Tested Topics

  1. Identify tautology/contradiction — Use falsification method
  2. Count satisfying assignments — Build truth table, count T rows
  3. Apply De Morgan's and Material Implication — Simplification questions
  4. Convert to CNF — Follow 4-step algorithm
  5. Resolution proof — Negate goal, convert to clauses, resolve to □
  6. Identify valid inference — Modus Ponens/Tollens vs fallacies
  7. Complexity: 2-SAT vs 3-SAT — P vs NP boundary
  8. Horn clause identification — Count positive literals ≤ 1

10. Resolution Proof Quick Steps

📝 Resolution Checklist

  1. ✅ Convert KB to CNF
  2. ✅ Negate the goal and convert to clauses
  3. ✅ Add negated goal clauses to KB clauses
  4. ✅ Find pairs with complementary literals
  5. ✅ Resolve and add resolvents
  6. ✅ If empty clause □ derived → proved!
  7. ✅ If no new resolvents possible → not provable
