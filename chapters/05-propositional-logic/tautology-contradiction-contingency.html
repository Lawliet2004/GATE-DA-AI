Tautologies, Contradictions, and Contingencies

GATE DA — High Importance

🎯 Learning Objectives

  • Distinguish between tautologies, contradictions, and contingencies
  • Test a formula's classification using truth tables
  • Understand the concept of satisfiability
  • Know the important tautologies tested in GATE

1. Definitions

Definition: Tautology

A propositional formula α is a tautology if it is true under every possible interpretation. We write ⊨ α.

⊨ α ⟺ v(α) = True for all interpretations v

Definition: Contradiction

A propositional formula α is a contradiction (also called unsatisfiable) if it is false under every possible interpretation.

α is a contradiction ⟺ v(α) = False for all interpretations v

Definition: Contingency

A propositional formula α is a contingency if it is neither a tautology nor a contradiction — it is true under some interpretations and false under others.

Definition: Satisfiable

A formula α is satisfiable if there exists at least one interpretation under which it is true. A formula is satisfiable if and only if it is not a contradiction.

Satisfiable ⟺ NOT a contradiction

Key Relationships

  • α is a tautology ⟺ ¬α is a contradiction
  • α is a contradiction ⟺ ¬α is a tautology
  • α is a tautology ⟹ α is satisfiable
  • α is a contingency ⟹ α is satisfiable
  • Tautology ∪ Contingency = Satisfiable formulas

3. Satisfiability

The concept of satisfiability is central to computational logic and is the foundation of the famous SAT problem.

Classification Summary

CategoryTrue in how many interpretations?Satisfiable?
TautologyAll 2nYes
ContingencySome (1 to 2n−1)Yes
ContradictionNone (0)No

Theorem: Satisfiability and Entailment

α ⊨ β (α entails β) if and only if α ∧ ¬β is unsatisfiable (a contradiction).

α ⊨ β ⟺ (α ∧ ¬β) is unsatisfiable

This is the theoretical basis for proof by refutation, used in the resolution method.

5. Quick Tests Without Full Truth Tables

📝 Shortcut Methods

  • To check if contradiction: Try to find an assignment making it true. If you can't, it's a contradiction.
  • To check if tautology: Try to find an assignment making it false. If you can't, it's a tautology.
  • For implications α → β: Set α = T and β = F. Check if this leads to a contradiction in variable assignments. If yes, the formula is a tautology.

Quick Test: Is ((P → Q) ∧ P) → Q a tautology?

Try to make it false: set the outer → to false.

  1. For outer → to be F: antecedent must be T and consequent must be F.
  2. So Q = F, and (P → Q) ∧ P = T.
  3. P = T (from P = T in the conjunction).
  4. P → Q = T → F = F. But we need (P → Q) = T. Contradiction!

We cannot make the formula false, so it is a tautology. ✅

6. GATE Exam Tips

🎯 Exam Strategy

  • For MCQs asking "which is a tautology?", try to falsify each option — the one you can't falsify is the answer.
  • For NAT questions asking "how many satisfying assignments?", build the truth table and count T rows.
  • Remember: Tautology negated = Contradiction. Use this to quickly classify negations.
  • The number of satisfying assignments of a tautology with n variables is exactly 2n.
