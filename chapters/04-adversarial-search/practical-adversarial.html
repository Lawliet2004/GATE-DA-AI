2. How the AI Works

Minimax for Tic-Tac-Toe

The AI explores the entire remaining game tree from the current position. Since Tic-Tac-Toe is small enough (max ≈9! = 362,880 states), minimax can solve it completely without any cutoff depth. The AI plays perfectly and will never lose.

2.2 Why the AI Never Loses

The minimax value of Tic-Tac-Toe from the initial state is 0 (draw). With perfect play by both sides, neither player can win. Since the AI always plays perfectly, the best you can achieve is a draw.

3. Experiments to Try

  • Experiment 1: Play with alpha-beta ON, note total nodes. Reset and play the same moves with alpha-beta OFF. Compare.
  • Experiment 2: Try to win! You’ll find the AI always blocks threats and forces a draw.
  • Experiment 3: Start in the corner vs. center vs. edge. Notice how the AI responds differently and the node counts change.

4. Algorithm Applied

Minimax for Tic-Tac-Toe (Simplified)
function bestMove(board, useAlphaBeta):
    bestScore ← −∞
    bestAction ← null
    for each empty cell c in board:
        board[c] ← 'O'  // AI move
        if useAlphaBeta:
            score ← alphabeta(board, false, −∞, +∞)
        else:
            score ← minimax(board, false)
        board[c] ← empty
        if score > bestScore:
            bestScore ← score
            bestAction ← c
    return bestAction

// Utility: X wins → −1, O wins → +1, Draw → 0
